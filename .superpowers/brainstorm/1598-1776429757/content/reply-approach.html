A — Hover/Tap → Reply Button (↩)

Tombol ↩ muncul saat hover (desktop) atau long-press (mobile). Quote preview muncul di atas input, bisa di-dismiss. Bubble reply berisi blok quote di atas teks.

B — Tombol ↩ Selalu Visible

Tidak perlu hover. Tombol reply selalu muncul di samping bubble. Lebih accessible di mobile. Tapi lebih ramai secara visual.

C — Context Menu (Long-Press / Klik Kanan)

Long-press bubble → popup menu dengan pilihan: Reply, Copy, (hapus jika milik sendiri). Bersih secara visual. Tapi lebih banyak step untuk user.
