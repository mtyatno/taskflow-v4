# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Tambah dark mode (Charcoal palette) dengan toggle switch ☀️/🌙 di top bar, preferensi tersimpan di localStorage.

**Architecture:** Semua styling pakai CSS variables di `:root`. Dark mode hanya perlu override variables tersebut via `[data-theme="dark"]` selector di `<html>`. Toggle state di-manage React (`useState`) dan di-sync ke `document.documentElement` + `localStorage`.

**Tech Stack:** React 18 (UMD), CSS variables, localStorage, single-file SPA (`static/index.html`)

## Task 2: Inisialisasi Theme dari localStorage

**Files:**
- Modify: `static/index.html` — tepat sebelum tag `<script type="text/babel">`

- [ ] **Step 1: Tambah script inisialisasi theme**

  Cari tag `<script type="text/babel">` (sekitar baris 151), tambahkan tepat SEBELUMNYA:

  ```html
  <script>
    (function() {
      var t = localStorage.getItem('tf_theme') || 'light';
      document.documentElement.setAttribute('data-theme', t);
    })();
  </script>
  ```

  Menggunakan IIFE agar tidak ada variable global. Dijalankan sebelum React render sehingga tidak ada flash of unstyled content (FOUC).

- [ ] **Step 2: Verifikasi FOUC prevention**

  Di DevTools → Application → Local Storage → set `tf_theme` = `dark`, lalu reload halaman. Halaman harus langsung muncul dalam dark mode tanpa flip ke light dulu.

- [ ] **Step 3: Commit**

  ```bash
  git add static/index.html
  git commit -m "feat: initialize theme from localStorage before React render (no FOUC)"
  ```

## Task 3: Komponen ThemeToggle

**Files:**
- Modify: `static/index.html` — setelah baris `// ── Toast component` (sekitar baris 180)

- [ ] **Step 1: Tambah komponen ThemeToggle**

  Cari `// ── Toast component`, tambahkan setelah komponen `Toast` selesai (setelah penutup `}`):

  ```jsx
  // ── Theme Toggle ────────────────────────────────────────────
  function ThemeToggle({ theme, onToggle }) {
    const isDark = theme === 'dark';
    return (
      <div style={{ display: 'flex', alignItems: 'center', gap: 6, cursor: 'pointer' }} onClick={onToggle} title={isDark ? 'Switch to Light' : 'Switch to Dark'}>
        <span style={{ fontSize: 14, lineHeight: 1 }}>☀️</span>
        <div style={{
          width: 36, height: 20, borderRadius: 10, position: 'relative', transition: 'background 0.2s',
          background: isDark ? 'var(--accent)' : '#d1d5db',
        }}>
          <div style={{
            position: 'absolute', top: 2, width: 16, height: 16, borderRadius: '50%',
            background: 'white', boxShadow: '0 1px 3px rgba(0,0,0,0.3)',
            transition: 'left 0.2s',
            left: isDark ? 18 : 2,
          }} />
        </div>
        <span style={{ fontSize: 14, lineHeight: 1 }}>🌙</span>
      </div>
    );
  }
  ```

## Task 4: Wiring Theme State di App + Top Bar

**Files:**
- Modify: `static/index.html` — fungsi `App()` dan desktop/mobile top bar

- [ ] **Step 1: Tambah `theme` state di App**

  Cari `function App() {` lalu temukan baris `const [user, setUser] = useState(null);` (sekitar baris 3835). Tambahkan setelah baris itu:

  ```js
  const [theme, setTheme] = useState(
    () => localStorage.getItem('tf_theme') || 'light'
  );

  const toggleTheme = useCallback(() => {
    setTheme(prev => {
      const next = prev === 'light' ? 'dark' : 'light';
      document.documentElement.setAttribute('data-theme', next);
      localStorage.setItem('tf_theme', next);
      return next;
    });
  }, []);
  ```

- [ ] **Step 2: Sisipkan ThemeToggle di desktop top bar**

  Cari blok desktop top bar (sekitar baris 4454):
  ```jsx
  <div data-notif-panel style={{ display: "flex", alignItems: "center", gap: 10, position: "relative" }}>
  ```
  Tambahkan `<ThemeToggle>` tepat SEBELUM tombol `＋ Tambah Task`:
  ```jsx
  <ThemeToggle theme={theme} onToggle={toggleTheme} />
  ```
  Sehingga urutan menjadi: 🔔 Bell → ThemeToggle → ＋ Tambah Task

- [ ] **Step 3: Sisipkan ThemeToggle di mobile top bar**

  Cari blok mobile top bar (`className="mobile-topbar"`, sekitar baris 4436). Tambahkan `<ThemeToggle>` di dalamnya, setelah tombol menu (☰):
  ```jsx
  <ThemeToggle theme={theme} onToggle={toggleTheme} />
  ```
